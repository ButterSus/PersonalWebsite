## Билет 10.

### 10.1. Второй закон Ньютона

Произведение массы тела на вектор ускорения тела равно вектору равнодействующей сил,
действующих на тело.

<procedure>

```tex
\begin{align}
&\vec{F} = m\vec{a}&\\
&\vec{F}\uparrow\uparrow\vec{a}&
\end{align}
```

Вектор силы сонаправлен с вектором ускорения, так как масса - это положительная величина.

</procedure>

Второй закон Ньютона работает только в **инерциальных системах отсчёта**.

### 10.2. Вывод закона движения материальной точки под действием постоянной силы

Мы можем получить закон движения материальной точки для равномерного и равнопеременного движения,
проинтегрировав второй закон Ньютона.

<procedure>

```tex
\begin{align}
&\vec{F} = m\vec{a}\ _\texttt{(второй закон Ньютона)}&\\
&\Leftrightarrow m\frac{d^2\vec{r}}{dt^2} = \vec{F}&\\
&\Rightarrow \frac{d^2\vec{r}}{dt^2} = \frac{\vec{F}}{m} = \vec{a}\ _\texttt{(проинтегрируем)}&\\
&\Rightarrow \frac{d\vec{r}}{dt} = \vec{v}_0 + \vec{a}t\ _\texttt{(проинтегрируем)}&\\
&\Rightarrow \vec{r} = \vec{r}_0 + \vec{v}_0t + \frac{\vec{a}t^2}{2}\ _\texttt{(есть закон движения)}&
\end{align}
```

</procedure>

### 10.3. Третий закон Ньютона

Два тела действуют друг на друга с силами, равными по модулю,
противоположными по направлению.

Эти силы имеют одну и ту же физическую природу и направлены
вдоль одной прямой, соединяющей их точки приложения.

<procedure>

<img src="третий_закон_ньютона.png" width="200" alt="Третий закон Ньютона"/>

```tex
\begin{align}
&\vec{F}_{12} = -\vec{F}_{21}&\\
&\vec{F}_{12}\uparrow\downarrow\vec{F}_{21}&
\end{align} 
```

</procedure>

### 10.5. Поступательные и центробежные силы инерции

В неинерциальной системе отсчёта существуют силы, которые не существуют в инерциальной системе отсчёта.

Они нужны для того, чтобы в неинерциальной системе отсчёта выполнялся второй закон Ньютона.

<procedure>

Существуют два типа сил инерции:

1. При поступательном движении тела относительно неинерциальной системы отсчёта
   существует сила инерции, направленная противоположно ускорению тела.

    ```tex
    \vec{F}_\texttt{поступательное} = -m\vec{a}_\texttt{переносная} 
    ```

2. При вращательном движении тела относительно неинерциальной системы отсчёта
   существует **центробежная сила** инерции, направленная от центра вращения.

    ```tex
    \vec{F}_\texttt{центробежная} = m\omega^2\vec{r}_\perp
    ```

    <img src="центробежная_сила.png" width="300" alt="Центробежная сила"/>

</procedure>

## Билет 11.

### 11.1. Закон всемирного тяготения

Любые два материальных тела притягиваются друг к другу с силой, прямо пропорциональной
произведению их масс и обратно пропорциональной квадрату расстояния между ними.

<procedure>

```tex
\begin{align}
&\vec{F} = -G\frac{m_1m_2}{r^2}\vec{e}_r&\\
&\vec{F}\uparrow\downarrow\vec{e}_r&
\end{align}
```

</procedure>

Гравитационная постоянная - это коэффициент пропорциональности в законе всемирного тяготения.

<procedure>

```tex
\begin{align}
&G = 6.67 \cdot 10^{-11}\frac{\texttt{м}^3}{\texttt{кг} \cdot \texttt{с}^2}&

```

</procedure>

### 11.3. Сила тяжести

Сила тяжести - это сила гравитационного притяжение,
действующая на тело со стороны планеты (Земли).

<procedure>

```tex
\begin{align}
&\vec{F}_\texttt{тяж} = m\vec{g} = G\frac{mM}{R^2}\vec{e}_R&\\
&\Rightarrow 1.\ GM = gR^2&\\
&\Rightarrow 2.\ g = G\frac{M}{R^2}&
\end{align}
```

</procedure>

### 11.4. Зависимость силы тяжести от высоты

Сила тяжести убывает в геометрической прогрессии с увеличением высоты.

<procedure>

```tex
\begin{align}
&\Rightarrow 3.\ mg(h) = G\frac{mM}{(R + h)^2}&\\
&\Leftrightarrow g(h) = G\frac{M}{(R + h)^2}&
\end{align}
```

</procedure>

### 11.5. Вес тела, невесомость и перегрузки

<emphasis>Вес тела</emphasis> - это сила, с которой тело действует на опору.

<procedure>

```tex
\begin{align}
&\vec{F}_\texttt{вес} = P = m\vec{g}&\\
&\vec{F}_\texttt{вес}\uparrow\uparrow\vec{g}&\\\\
&\textbf{По третьему закону Ньютона:}&\\
&\vec{F}_\texttt{вес} = -\vec{F}_\texttt{опоры}&\\
\end{align}
```

<emphasis>Невесомость</emphasis> - это когда вес тела меньше силы тяжести.

<emphasis>Перегрузка</emphasis> - это когда вес тела больше силы тяжести.

</procedure>

### 11.6. Вес тела, находящегося на поверхности Земли, и его зависимость от географической широты.

<procedure>

<img src="географическая-широта.png" width="300" alt="Географическая широта"/>

```tex
\begin{align}
&P = \vec{F}_\texttt{вес} + \vec{F}_\texttt{ц.б.}&\\
&\vec{F}_\texttt{ц.б.} = m\omega^2\vec{e}_r&\\
&\Rightarrow \underline{P = m\vec{g} + m\omega^2R_{\texttt{земли}}\vec{e}_r}&
\end{align}
```

</procedure>

## Билет 12.

### 12.1. Движение искусственных спутников Земли

<procedure>

<img src="движение-искусственных-спутников.png" width="200" alt="Движение искусственных спутников"/>

Из рисунка видно, что радиус есть сумма радиуса Земли и высоты спутника.

```tex
\begin{align}
&\vec{F} = G\frac{mM}{(R + h)^2}\vec{e}_r&\\
&a=\frac{v^2}{R+h}&\\\\
&\texttt{По второму закону Ньютона:}&\\
&a=\frac{F}{m} = G\frac{M}{(R+h)^2}&\\\\
&\Rightarrow \frac{v^2}{R+h} = G\frac{M}{(R+h)^2}&\\
&\Rightarrow v = \sqrt{G\frac{M}{R+h}}&
\end{align}
```

</procedure>

### 12.2. Первая космическая скорость

<emphasis>Первая космиеская скорость</emphasis> - это минимальная скорость, с которой
нужно запустить тело, чтобы оно не падало на Землю.

<procedure>

```tex
\begin{align}
&\texttt{При } h = 0 \texttt{ получаем:}&\\
&v_1 = \sqrt{G\frac{M}{R}} = 7.9\frac{\texttt{км}}{\texttt{с}}&\\
&\Leftrightarrow v_1=\sqrt{gR}\ \Leftarrow GM = gR^2&
\end{align}
```

Это минимальная скорость, так как **высота спутника не может быть меньше нуля**,
а скорость **прямо пропорциональна** квадратному корню из суммы радиуса Земли и **высоты спутника**.

</procedure>

### 12.3. Законы Кеплера

#### 12.3.1. Первый закон Кеплера

<emphasis>Первый закон Кеплера</emphasis> - все планеты движутся по эллипсам,
в одном из фокусов которых находится Солнце.

<procedure>

Так происходит потому, что Солнце имеет невероятно большую массу,
не сравнимую с массой планет, поэтому можно считать, что Солнце неподвижно.

<img src="первый_закон_кеплера.png" height="300" alt="Первый закон Кеплера"/>

</procedure>

#### 12.3.2. Второй закон Кеплера

<emphasis>Второй закон Кеплера</emphasis> - радиус-вектор, проведённый из Солнца к планете,
за равные промежутки времени, заметает равные площади.

<procedure>

Доказательство не требуется, т.к. **это дичь!**

<img src="второй_закон_кеплера.png" height="300" alt="Второй закон Кеплера"/>

</procedure>

#### 12.3.3. Третий закон Кеплера

<emphasis>Третий закон Кеплера</emphasis> - квадраты периодов обращения планет
пропорциональны кубам больших полуосей их орбит.

<procedure>

<img src="третий_закон_кеплера.png" height="300" alt="Третий закон Кеплера"/>

```tex
\begin{align}
&\texttt{Пусть } T_1 \texttt{ и } T_2 \texttt{ - периоды обращения планет}&\\
&\texttt{Пусть } a_1 \texttt{ и } a_2 \texttt{ - большие полуоси орбит}&\\
&\texttt{Тогда:}&\\
&\frac{T_1^2}{T_2^2} = \frac{a_1^3}{a_2^3}\ - \texttt{для эллипсов}&
\end{align}
```

</procedure>

<procedure title="Вывод" collapsible="true">

```tex
\begin{align}
&G\frac{mM}{r^2} = m\underset{=a_{\texttt{ц.с.}}}{\underline{\frac{v^2}{r}}}&\\
&\Rightarrow \left[\begin{aligned}
G\frac{M}{r} = v^2\\
v = \frac{2\pi r}{T}
\end{aligned}\right .&\\
&\Rightarrow G\frac{M}{r} = \frac{4\pi^2r^2}{T^2}&\\
&\Rightarrow T^2 = \frac{4\pi^2r^3}{GM}&\\\\
&\Rightarrow \frac{T_1^2}{T_2^2} = \frac{r_1^3}{r_2^3}\ - \texttt{для окружностей}&
\end{align}
```

</procedure>

### 13.1. Силы упругости

<emphasis>Силы упругости</emphasis> - это силы, возникающие в теле при
сжатии или растяжении тела.

Причём силы упругости направлены так, чтобы вернуть тело в исходное состояние и возникают

### 13.2. Упругие и пластические деформации

<emphasis>Упругие деформации</emphasis> - это деформации, которые исчезают после прекращения действия силы.

<emphasis>Пластические деформации</emphasis> - это деформации, которые не исчезают после прекращения действия силы.

<procedure>

<img src="упругая-пластическая-деформации.png" width="400" alt="Упругая и пластическая деформации"/>

</procedure>

### 13.3. Закон Гука

<emphasis>Закон Гука</emphasis> - деформация тела прямо пропорциональна силе, вызвавшей эту деформацию.

<procedure>

<img src="закон_гука.png" height="300" alt="Закон Гука"/>

```tex
\begin{align}
&F_\texttt{упр} = -kx&\\
&F_\texttt{упр}\uparrow\downarrow F&\\
&\texttt{где:}&\\
&k - \texttt{коэффициент упругости}&\\
&x - \texttt{длина удлинения}&
\end{align}
```

</procedure>

### 13.4. Модуль Юнга и его взаимосвязь с коэффициентом упругости

<emphasis>Модуль Юнга</emphasis> - это модуль упругости первого рода, который характеризует
упругие свойства тела при одноосном растяжении.

<procedure>

```tex
\begin{align}
&\texttt{По закону Гука:}&\\
&F_\texttt{упр} = -kx&\\
&\texttt{По определению модуля Юнга:}&\\
&k = \frac{F_\texttt{упр}}{x} = \frac{ES}{l}&\\
&\texttt{где:}&\\
&E - \texttt{модуль Юнга}&\\
&S - \texttt{площадь поперечного сечения}&\\
&l - \texttt{длина}&\\\\
&\Rightarrow F_\texttt{упр} = -\frac{ES}{l}x&\\
\end{align}
```

</procedure>

<tip>

Мы обычно пренебрегаем изменением коэффициента упругости при упругих деформациях,
так как длина тела обычно много больше длины удлинения.

</tip>

### 13.5. Физический смысл модуля Юнга

Модуль Юнга показывает, с какой силой надо сжать или растянуть стержень
единичного поперечного сечения, чтобы получилось изменение длины на единицу.

<note>

Замечали ли вы, что тем больше модуль Юнга, тем труднее сжать или растянуть стержень?

</note>

### 14.2. Виды сил трения

<tldr>

<emphasis>Сухое трение</emphasis> - это трение, возникающее при соприкосновении твёрдых тел.

</tldr>

Существует два вида сил сухого трения:

1. <emphasis>Сила трения покоя</emphasis> - сила трения, возникающая при отсутствии относительного движения тел.

   <procedure>

   <img src="сила_трения_покоя.png" width="300" alt="Сила трения покоя"/>

   ```tex
   \begin{align}
   &F_\texttt{тр} \leqslant \mu N&\\
   &F_\texttt{тр} = -F&
   \end{align} 
   ```

   </procedure>

2. <emphasis>Сила трения скольжения</emphasis> - сила трения, возникающая при относительном движении тела и опоры.

   <procedure>

   <img src="сила_трения_скольжения.png" width="300" alt="Сила трения скольжения"/>

   ```tex
   \begin{align}
   &F_\texttt{тр} = \mu N&
   \end{align}
   ```

   </procedure>

### 14.3. Коэффициент трения

<emphasis>Коэффициент трения</emphasis> - это безразмерная величина, характеризующая силу трения.

<procedure>

```tex
\begin{align}
&F_\texttt{тр} = \mu N&\\
&\texttt{где:}&\\
&\mu - \texttt{коэффициент трения}&\\
&N - \texttt{сила реакции опоры}&
\end{align}
```

</procedure>

### 14.4. Направление сил сухого трения

Силы сухого трения направлены противоположно относительному движению тел.

<img src="направление_силы_трения.png" width="300" alt="Направление силы трения"/>

### 14.5. Зависимость силы сухого трения от внешней силы

В случае силы трения покоя сила трения не может превышать силу, приложенную к телу.

По этой причине, сила трения покоя возрастает с увеличением внешней силы, пока не достигнет максимального значения.

<procedure>

```tex 
\begin{align}
&F_\texttt{тр} \leqslant \mu N&\\
&F_\texttt{тр} = -F&
\end{align}
```

</procedure>

В случае если оно превышает максимальное значение, тело начинает двигаться, а сила трения становится
равной силе трения скольжения.

<procedure>

```tex
\begin{align}
&F_\texttt{тр} = \mu N&\\
&F_\texttt{тр} \neq -F&
\end{align}
```

</procedure>

Для лучшего понимания, посмотрите на график зависимости силы трения от внешней силы.

### 14.6. Вязкое трение

<emphasis>Вязкое трение</emphasis> - это сила трения, возникающая при движении тела в жидкости или газе.

<procedure>

<img src="вязкое_трение.png" width="300" alt="Вязкое трение"/>

</procedure>

<note>

Так, например, к вязкому трению можно отнести силу сопротивления воздуха.

</note>

<procedure title="Зависимость силы вязкого трения от скорости" collapsible="true">

```tex
\begin{align}
&\textbf{При малых скоростях:}&\\
&F_\texttt{тр} = \alpha v&\\

&F_\texttt{тр} = \beta v^2&\\
&F_\texttt{тр} \sim v^2&\\

&\beta v^2 = mg&\\
&\Rightarrow v = \sqrt{\frac{mg}{\beta}}&
\end{align}
```

</procedure>

## Билет 15.

### 15.1. Импульс материальной точки

<emphasis>Импульс материальной точки</emphasis> - это векторная величина,
равная произведению массы материальной точки на её скорость.

<procedure>

```tex
\begin{align}
&\vec{p} = m\vec{v}&\\
&\vec{p}\uparrow\uparrow\vec{v}&
\end{align}
```

</procedure>

<note>

Специальной единицей измерения импульса не существует, поэтому импульс измеряется в килограммах на метр в секунду.

</note>

### 15.2. Изменение импульса

Есть разность между импульсом в конечный момент времени и импульсом в начальный момент времени.

<procedure>

```tex
\begin{align}
&\Delta\vec{p} = \vec{p} - \vec{p}_0 = \vec{F} \Delta t&\\
\end{align}
```

</procedure>

### 15.3. Второй закон Ньютона в импульсной форме

Простым дифференцированием можно прийти к следующему выражению:

<procedure>

```tex
\begin{align}
&m\vec{a} = \vec{F}&\\
&\Leftrightarrow m\frac{d\vec{v}}{dt} = \vec{F}\Leftrightarrow \frac{dm\vec{v}}{dt} = \vec{F}&\\
&\Leftrightarrow \frac{d\vec{p}}{dt} = \vec{F} \Rightarrow \underline{\frac{\Delta\vec{p}}{\Delta t} = \vec{F}}&
\end{align}
```

</procedure>

### 15.4. Импульс силы

<emphasis>Импульс силы</emphasis> - это разность между импульсом в конечный момент
времени и импульсом в начальный момент времени.

<procedure>

```tex
\begin{align}
&\vec{I} = \vec{p} - \vec{p}_0 = \vec{F} \Delta t&\\
\end{align}
```

</procedure>

<tip>

Как следствие из импульса силы, ясно, что импульс системы материальных точек не меняется
без внешних сил.

</tip>

### 15.5. Импульс системы материальных точек

<tldr>

Силы, действующие в системе, делятся на внутренние и внешние.

1. <emphasis>Внутренние силы</emphasis> - это силы, действующие в системе, которые
   не изменяют импульс системы. _(т.к. по третьему закону Ньютона силы равны по модулю и противоположно направлены)_

2. <emphasis>Внешние силы</emphasis> - это силы, действующие в системе, которые
   изменяют импульс системы.

</tldr>

Начнём с простой системы из двух материальных точек.

<procedure>

```tex
\begin{align}
&\vec{p} = \vec{p}_1 + \vec{p}_2&\\
\end{align}
```

<img src="система_двух_тел.png" height="120" alt="Система двух тел"/>

Добавим внутренние силы, действующие в системе.

```tex
\begin{align}
&\begin{cases}
\frac{d\vec{p}_1}{dt} = \vec{F}_1 + \vec{T}_1\\
\frac{d\vec{p}_2}{dt} = \vec{F}_2 + \vec{T}_2\\
\vec{T}_1 = -\vec{T}_2\ - \texttt{по третьему закону Ньютона}
\end{cases}&\\
&\Rightarrow \frac{d\vec{p}_1}{dt} + \frac{d\vec{p}_2}{dt} = \vec{F}_1 + \vec{F}_2 + \vec{T}_1 + \vec{T}_2&\\
&\Rightarrow \frac{d\vec{p}}{dt} = \vec{F}_\texttt{внеш}
\end{align}
```

Таким образом, **скорость изменения импульса системы тел есть равнодействующая
внешних сил, приложенных к системе**.

</procedure>

Проведя аналогичные рассуждения несложно понять, что и импульс системы материальных точек
будет иметь следующий вид:

<procedure>

```tex
\begin{align}
&\vec{p} = \sum_{i=1}^n \vec{p}_i = \sum_{i=1}^n m_i\vec{v}_i&\\
&\Rightarrow \frac{dp}{dt} = \sum_{i=1}^n \frac{d\vec{p}_i}{dt} = \sum_{i=1}^n \vec{F}_i
+ \sum_{i=1}^n\sum_{i<j}^n \vec{T}_{ij}&\\
&\Rightarrow \frac{dp}{dt} = \vec{F}_\texttt{внеш} + \vec{T}_\texttt{внутр}^{\rightarrow0}
= \vec{F}_\texttt{внеш}&
\end{align}
```

</procedure>

### 15.6. Закон сохранения импульса

<emphasis>Закон сохранения импульса</emphasis> - это закон, согласно которому
**импульс замкнутой системы материальных точек с течением времени не меняется**.

В этом легко убедиться, так как скорость изменения импульса системы материальных точек
равна нулю, если в системе нет внешних сил.

<procedure>

```tex
\begin{align}
&\frac{dp}{dt} = \vec{F}_\texttt{внеш} = 0&\\
&\Rightarrow \vec{p} = \texttt{const}&
\end{align}
```

</procedure>

Подобные рассуждения можно проводить и **отдельно с осями координат**.

### 15.7. Закон сохранения проекции импульса

<emphasis>Закон сохранения проекции импульса</emphasis> получается из закона сохранения импульса.

Для примера рассмотрим закон сохранения проекции импульса на ось Ox.

<procedure>

```tex
\begin{align}
&\frac{dp_x}{dt} = F_x&\\
&\Rightarrow p_x = \texttt{const}&
\end{align}
```

</procedure>

## Билет 16.

### 16.1. Центр масс системы материальных точек

<emphasis>Центр масс системы материальных точек</emphasis> - это воображаемая точка
(мы не придумываем её, она просто не закреплена каким-то конкретным телом), которая
движется так, как если бы вся масса системы была сосредоточена в этой точке.

<procedure>

<img src="центр_масс.png" width="300" alt="Центр масс"/>

</procedure>

### 16.2. Импульс твёрдого тела

Твёрдое тело состоит из нескольких материальных точек, поэтому импульс твёрдого тела
есть сумма импульсов всех материальных точек, из которых состоит тело.

<procedure title="Вывод">

```tex
\begin{align}
&\vec{p} = \vec{p}_1 + \vec{p}_2 + \dots + \vec{p}_N = m_1\vec{v}_1 + m_2\vec{v}_2 + \dots + m_N\vec{v}_N&\\
&\frac{d\vec{p}}{dt} = \vec{F}_1 + \vec{F}_2 + \dots + \vec{F}_N = \vec{F}_\texttt{внеш}&\\
&= m_1\dot{\vec{v}}_1 + m_2\dot{\vec{v}}_2 + \dots + m_N\dot{\vec{v}}_N
= \sum_{i=1}^nm_i\vec{a}_i&\\
&\Rightarrow \sum_{i=1}^nm_i\vec{a}_i = \vec{F}_\texttt{внеш}&\\
\end{align} 
```

</procedure>

### 16.3. Способ вычисления координат центра масс

В векторной форме центр масс можно вычислить по следующей формуле:

<procedure>

```tex
\begin{align}
&\vec{r}_c = \frac{m_1\vec{r}_1 + m_2\vec{r}_2 + \dots + m_N\vec{r}_N}{m_1 + m_2 + \dots + m_N}
= \frac{\sum_{i=1}^nm_i\vec{r}_i}{\sum_{i=1}^nm_i}=\frac{1}{M}\sum_{i=1}^nm_i\vec{r}_i
\end{align}
```

</procedure>
 
Аналогично можно вывести для проекций на оси координат.

### 16.4. Теорема о движении центра масс

<emphasis>Теорема о движении центра масс</emphasis> - это теорема, согласно которой
**ускорение центра масс системы материальных точек равно векторной сумме всех внешних сил, действующих на систему**.

<procedure title="Вывод">

```tex
\begin{align}
&\vec{p} = \vec{p}_1 + \vec{p}_2 + \dots + \vec{p}_N = m_1\vec{v}_1 + m_2\vec{v}_2 + \dots + m_N\vec{v}_N&\\
&\frac{d\vec{p}}{dt} = \vec{F}_1 + \vec{F}_2 + \dots + \vec{F}_N = \vec{F}_\texttt{внеш}&\\
&\Rightarrow \sum_{i=1}^nm_i\vec{a}_i = \vec{F}_\texttt{внеш}&\\
&\Leftrightarrow \vec{r}_c = \frac{m_1\vec{r}_1 + m_2\vec{r}_2 + \dots + m_N\vec{r}_N}{m_1 + m_2 + \dots + m_N}&\\
&\Leftrightarrow \dot{\vec{r}}_c = \vec{v}_c
= \frac{m_1\vec{v}_1 + m_2\vec{v}_2 + \dots + m_N\vec{v}_N}{m_1 + m_2 + \dots + m_N}
= \frac{1}{M}\sum_{i=1}^nm_i\vec{v}_i&\\
&\Rightarrow \sum_{i=1}^nm_i\vec{a}_i = M\vec{v}_c&\\
&\Rightarrow M\vec{a}_c = \vec{F}_\texttt{внеш}\ -\ \texttt{теорема о движении центра масс}&\\\\
&\texttt{где:}&\\
&M = m_1 + m_2 + \dots + m_N - \texttt{масса системы}&\\
&\vec{r}_c - \texttt{радиус-вектор центра масс}&\\
&\vec{v}_c - \texttt{скорость центра масс}&\\
&\vec{a}_c - \texttt{ускорение центра масс}&
\end{align}
```

</procedure>

Как следствие из теоремы о движении центра масс, ясно, что если в системе нет внешних сил,
то центр масс системы материальных точек движется равномерно и прямолинейно.

```tex
\begin{align}
&\texttt{В этом примере}\ \vec{F}_p\ -\ \texttt{реактивная сила}&
\end{align}
```

### 17.2. Уравнение Мещерского

<emphasis>Уравнение Мещерского</emphasis> - гласит о следующем:

<procedure>

```tex
\begin{align}
&m\frac{d\vec{v}}{dt} = \sum\vec{F}_\texttt{внеш} + \vec{F}_{\texttt{реакт}}&
\end{align} 
```

</procedure>

<procedure title="Вывод">

<img src="мещерский_уравнение_1.png" height="180" alt="Уравнение Мещерского"/>

```tex
\begin{align}
&\textbf{В начальный момент времени:}&\\
&\Rightarrow\vec{p}_\texttt{сист} = (m + dm_\texttt{отд})\vec{v} + dm_\texttt{пр}\vec{v}_\texttt{пр}&
\end{align}
```

<img src="мещерский_уравнение_2.png" height="250" alt="Уравнение Мещерского"/>

```tex
\begin{align}
&\textbf{После некоторого промежутка времени:}&\\
&\Rightarrow \vec{p}_\texttt{сист}+d\vec{p}_\texttt{сист} =
(m + dm_\texttt{пр})(\vec{v} + d\vec{v}) + dm_\texttt{отд}\vec{v}_\texttt{отд}&\\\\
&\textbf{Вычитаем оба уравнения:}&\\
&\Rightarrow d\vec{p}_\texttt{сист} = md\vec{v} - dm_\texttt{отд}
(\vec{v} - \vec{v}_\texttt{отд}) + dm_\texttt{пр}(\vec{v} - \vec{v}_\texttt{пр})
+ \overset{\rightarrow0\,_\texttt{т.к. две буквы}\,d}{\overline{(dm_\texttt{пр}d\vec{v})}}&\\
&\Rightarrow d\vec{p}_\texttt{сист} = md\vec{v} + dm_\texttt{пр}(\vec{v} 
- \vec{v}_\texttt{пр}) - dm_\texttt{отд}(\vec{v} - \vec{v}_\texttt{отд})&\\\\
&\textbf{Подставляем в уравнение системы:}&\\
&\vec{v}_\texttt{отд} = \vec{v} + \vec{u}_\texttt{отд}&\\
&\vec{v}_\texttt{пр} = \vec{v} + \vec{u}_\texttt{пр}&\\
&\textbf{По второму закону Ньютона:}&\\
&d\vec{p}_\texttt{сист} = \sum\vec{F}_\texttt{внеш}\cdot dt&\\\\
&\Rightarrow d\vec{p}_\texttt{сист} = md\vec{v} + dm_\texttt{отд}\vec{u}_\texttt{отд}
- dm_\texttt{пр}\vec{u}_\texttt{пр} = \sum\vec{F}_\texttt{внеш}\cdot dt\ |\ \cdot\frac{1}{dt}&\\
&\Rightarrow \underline{m\frac{d\vec{v}}{dt} = \sum\vec{F}_\texttt{внеш}
- \frac{dm_\texttt{отд}}{dt}\vec{u}_\texttt{отд} + \frac{dm_\texttt{пр}}{dt}\vec{u}_\texttt{пр}}&
\end{align}
```

</procedure>

### 17.3. Реактивная сила

<emphasis>Реактивная сила</emphasis> - это сила, возникающая при реактивном движении.

<procedure>

```tex
\begin{align}
&\textbf{Из уравнения Мещерского:}&\\
&\vec{F}_{\texttt{реакт}} = \frac{dm_\texttt{пр}}{dt}\vec{u}_\texttt{пр}
- \frac{dm_\texttt{отд}}{dt}\vec{u}_\texttt{отд} = \mu_\texttt{пр}\vec{u}_\texttt{пр}
- \mu_\texttt{отд}\vec{u}_\texttt{отд}&\\
\end{align}
```

</procedure>

## Билет 18.

### 18.1. Механическая работа

<emphasis>Механическая работа</emphasis> - это скалярная величина, равная
произведению векторов силы и перемещения тела.

Для работы есть своя единица измерения - джоуль.

<procedure>

<img src="определение_работы.png" height="200" alt="Определение работы"/>

```tex
\begin{align}
&A = \vec{F}\cdot\vec{S}&\\
&\texttt{где:}&\\
&A - \texttt{механическая работа}&\\
&\vec{F} - \texttt{сила}&\\
&\vec{S} - \texttt{перемещение}&\\\\
&\texttt{Единица измерения:}&\\
&\texttt{Джоуль (Дж)} = \texttt{Н}\cdot\texttt{м}&\\\\
&\texttt{В случае одномерного движения}&\\
&A = F\cdot S&\\\\
&\texttt{Работа равнодействующей силы:}&\\
&A = \sum_{i=1}^n A_i&
\end{align}
```

</procedure>

<note>

**Работа может быть как положительной, так и отрицательной.**

У работы много формулировок, так как она тесно связана с энергией.
К работе так же относят например силу тяжести, силу трения, силу упругости и т.д.

</note>

### 18.2. Относительность работы

<emphasis>Относительность работы</emphasis> - это тот факт, что работа приложенной силы
имеют одинаковую величину для всех инерциальных систем отсчёта.

Это есть [одно из следствий относительности Галилея](Physics-1.md#5-3).

### 18.3. Работа постоянной и переменной силы

<procedure>

```tex
\begin{align}
&\textbf{Постоянная сила:}&\\
&A = F\cdot \Delta x&\\\\
&\textbf{Переменная сила:}&\\
&A = \int_0^t F(t)\cdot (v\cdot dt)&
\end{align} 
```

</procedure>

### 18.4. Графический способ вычисления работы

Как пример можно привести **работу силы упругости**.
Рассмотрим график зависимости силы упругости от перемещения тела.

<procedure>

<img src="работа_силы_упругости.png" width="300" alt="Работа силы упругости"/>

```tex
\begin{align}
&A = \int_0^t -kx\cdot(v_x\cdot dt) = \int_{x_0}^x -kx\cdot dx = \frac{1}{2}k\Delta x^2&
\end{align}
```

</procedure>

### 18.5. Аддитивность работы

<emphasis>Аддитивность работы</emphasis> - это тот факт, что работа равнодействующей силы
равна сумме работ всех сил, действующих на тело.

<procedure>

```tex
\begin{align}
&A = A_1 + A_2 + \dots + A_n = \sum_{i=1}^n A_i&
\end{align}
```

</procedure>

### 18.6. Мощность постоянной и переменной силы

<emphasis>Мощность</emphasis> - это физическая величина, которая показывает
скорость выполнения работы и равна отношению работы к времени.

<procedure>

```tex
\begin{align}
&N = \frac{A}{t}&\\
&\texttt{где:}&\\
&P - \texttt{мощность}&\\
&A - \texttt{работа}&\\
&t - \texttt{время}&\\\\
&\texttt{Единица измерения:}&\\
&\texttt{Ватт (Вт)} = \frac{\texttt{Дж}}{\texttt{с}}&\\\\
&\texttt{Постоянная сила:}&\\
&\overline{N} = \frac{F\cdot \Delta x}{t} = F\cdot v&\\\\
&\texttt{Переменная сила:}&\\
&\overline{N} = \frac{\int_0^t F(t)\cdot (v\cdot dt)}{t} = \frac{A}{t}&\\
&N(t) = \frac{dA}{dt} = F\frac{dr}{dt}=F_\tau v(t)\ -\ \texttt{мгновенная мощность}&
\end{align}
```

</procedure>
